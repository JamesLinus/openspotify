ABOUT
=====
An attempt to write an ABI-compatible libspotify clone with
technology from the despotify project

Work in progress !! :)


LAYOUT
======
include/
- Only contains <spotify/api.h>

libopenspotify/
- The libspotify clone

openssl/
- A bundled OpenSSL library (0.9.8k) for building on Win32
